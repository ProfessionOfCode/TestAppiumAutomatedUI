# Technical Onboarding Documentation

## Project Overview

This project is designed to provide a comprehensive solution for UI testing
WinUI applications using Appium server.

It leverages WinUI 3, .NET8 to deliver a robust application
that meets the needs of professioners of code.

## Installation Instructions

1. **Prerequisites**:

- Ensure you have .NET 8 SDK [Browse all .NET versions to download | .NET](https://dotnet.microsoft.com/en-us/download/dotnet).

- Install Appium server following [Appium's GitHub page](https://github.com/appium/appium#installation).

- Make sure you have a compatible version of Node.js installed.
